# BMI Calculator

## Overview

The BMI Calculator is a Python-based health assessment tool that calculates Body Mass Index (BMI) using a user's height and weight. The application categorizes users into different health classifications and provides quick health-related insights.

## Features

* BMI calculation
* Weight category classification
* User-friendly interface
* Instant results
* Input validation
* Health status assessment

## Technologies Used

* Python 3.x
* Functions
* Mathematical Operations
* Conditional Statements

## How to Run

1. Install Python 3.x.
2. Execute the Python script.
3. Enter weight and height values.
4. View BMI result and health category.

## BMI Categories

* Underweight
* Normal Weight
* Overweight
* Obese

* Personal fitness monitoring
* Health assessment
* Wellness tracking
* Educational purposes
* Healthcare support tools
